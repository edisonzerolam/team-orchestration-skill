## 分析框架

### 第一步：宏观环境扫描

收集隔夜全球市场、重要经济数据发布、央行动态、政策动向、大宗商品异动等信息。

**数据获取**：
```bash
# 全球市场概览
westock-data market sh000001,sz399001,sz399006,hkHSI,us.IXIC,us.INX

# 投资日历（经济数据与央行事件）
westock-data calendar <today> 30 0 1

# 市场资讯（获取宏观新闻）
westock-data marketnews hs
westock-data marketnews us
```

如 westock-data 无法获取特定宏观数据，降级使用 WebSearch 搜索：
```
WebSearch: "中国CPI PPI最新数据 2026"
WebSearch: "美联储利率决议最新"
```

### 第二步：宏观周期定位

基于美林时钟框架，判断当前经济周期所处阶段：

| 阶段 | 经济 | 通胀 | 最优资产 | A股偏好 |
|------|------|------|---------|---------|
| 复苏 | ↑ | ↓ | 股票 | 成长、消费 |
| 过热 | ↑ | ↑ | 商品 | 周期、资源 |
| 滞胀 | ↓ | ↑ | 现金 | 防御、高股息 |
| 衰退 | ↓ | ↓ | 债券 | 利率敏感、科技 |

关键判断指标：
- 经济方向：PMI、工业增加值、社零、信贷脉冲
- 通胀方向：CPI、PPI、核心CPI
- 流动性：社融增速、M1-M2 剪刀差、DR007

### 第三步：传导路径拆解

将宏观变量逐层拆解到行业盈利和估值影响：

```
[宏观变量] → [中间变量] → [行业影响]
```

常见传导链条：
- 降息 → 无风险利率↓ → 长久期资产估值提升（科技/医药）
- 人民币贬值 → 出口竞争力↑ → 家电/纺织/机械出口受益
- 财政发力 → 基建投资↑ → 建材/钢铁/工程机械需求↑
- 大宗涨价 → PPI↑ → 上游资源盈利弹性（煤炭/有色/石化）
- 地产放松 → 销售回暖 → 建材/家居/银行资产质量改善

### 第四步：受益行业分层

按传导距离分层：

- **一阶受益**：宏观变量直接作用的行业（如降息→地产/券商）
- **二阶受益**：间接受益行业（如地产回暖→家居/建材/物业）
- **反向受损**：受宏观变量负面影响的行业

### 第五步：传导时滞与验证指标

每条传导路径标注：
- 预计传导时滞（即时 / 1-3个月 / 3-6个月 / 6个月以上）
- 可跟踪的验证指标（如社融增速、地产销售面积、PMI 新订单）
- 当前验证状态（已验证 / 待验证 / 未启动）

## 注意事项

- 宏观判断需标注数据来源（如 westock-data market、WebSearch 搜索词等）
- 传导路径需区分"逻辑推演"和"已有数据验证"，不要把假设当结论

- 如数据不足以支撑判断，明确标注"信息不充分，需进一步跟踪"
