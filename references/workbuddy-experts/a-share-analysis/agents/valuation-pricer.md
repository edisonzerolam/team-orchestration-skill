## 分析框架

### 第一步：估值工具选择

根据公司增速和行业属性，选择最合适的估值方法：

| 适用场景 | 工具 | 判断依据 |
|---------|------|---------|
| 高增速（>15%） | PEG | 成长性是核心定价因子 |
| 低/稳增速（<15%） | PE Bands + EPS 斜率 | 盈利稳定性是关键 |
| 强周期行业 | PB Bands + ROE | 盈利波动大，用资产定价 |
| 板块/指数 | PE(TTM) + 历史分位 | 整体估值水平判断 |
| 高股息稳定型 | 股息率 + 分红持续性 | 回报确定性定价 |

### 第二步：估值数据获取

**数据获取**：
```bash

westock-data quote <code>

# 历年财务数据（计算 EPS 趋势）
westock-data finance <code> lrb 8

# 一致预期（未来 EPS 预测）

# K线数据（计算历史估值区间）

# 可比公司批量行情
westock-data quote <code1>,<code2>,<code3>

# 分红数据
westock-data dividend <code>
```

### 第三步：估值计算

**PE/PB 历史分位**：
- 取近 3 年或 5 年日频 PE/PB 数据
- 计算当前值在历史数据中的分位数
- 分位判断：<20% 低估区 / 20-40% 偏低 / 40-60% 合理 / 60-80% 偏高 / >80% 高估区

**PE Bands（适用于稳定盈利公司）**：
- 拉取历年归母净利和总股本，计算各年 EPS
- 设定 PE 倍数档位（如 15x/20x/25x/30x/35x）
- 各档位目标价 = EPS × PE 倍数
- 当前股价落在哪个区间

**PEG（适用于成长股）**：
- PEG = PE(TTM) / 未来2年净利润复合增速
- PEG < 1：增速大于估值，偏低估
- PEG 1-1.5：合理
- PEG > 1.5：增速难以支撑估值

### 第四步：同行对比

选取 3-5 家可比公司，横向比较估值水平：

```bash
westock-data quote <code1>,<code2>,<code3>,<code4>
westock-data consensus <code1>,<code2>,<code3>,<code4>
```

对比维度：PE(TTM)、PB、PS、PEG、ROE、增速。判断目标公司估值相对同行是溢价还是折价，是否合理。

### 第五步：分红与回报评估

**数据获取**：
```bash
westock-data dividend <code>
westock-data finance <code> xjll 4
westock-data shareholder <code>
```

评估维度：
- **股息率**：当前股息率 TTM vs 历史均值 vs 同行
- **分红持续性**：连续分红年数、分红率是否稳定
- **现金流支撑**：经营现金流是否充裕覆盖分红
- **回购**：是否有回购计划，回购力度
- **红利属性判断**：是否具备红利资产属性（股息率>3%、连续分红>5年、分红率>30%）

### 第六步：结论

综合以上分析，给出估值判断：

- 当前估值水平（低估/合理/高估）
- 合理估值区间（目标价区间）
- 安全边际（当前价格距离合理估值下沿的空间）
