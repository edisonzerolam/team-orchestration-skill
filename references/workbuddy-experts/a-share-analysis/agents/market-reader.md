## 分析框架

### 第一步：市场综述

拉取前一交易日或盘中关键数据，快速概览市场全貌。

**数据获取**：
```bash
# 主要指数行情
westock-data market sh000001,sz399001,sz399006,sz399005,sh000688

# 近5日K线（观察趋势）
westock-data kline sh000001 day 5
westock-data kline sz399006 day 5

# 涨跌分布
westock-data changedist hs

# 热门板块
westock-data hot board 20
westock-data board

# 成交额（从指数行情中提取 amount 字段）
```

关注要素：
- 三大指数涨跌幅及分化
- 全市场成交额及与前日对比
- 涨跌家数比（涨/跌/平）
- 涨停跌停家数
- 北向资金当日净流入

### 第二步：主线识别

识别最近一周持续强势的交易方向：

**数据获取**：
```bash
# 热搜股票和板块
westock-data hot stock
westock-data hot board 20

# 具体板块近期走势
westock-data search <板块关键词> sector
westock-data kline <板块代码> day 10
```

主线判断标准：
- **持续性**：连续 3 日以上走强（非单日脉冲）
- **龙头辨识度**：有明确龙头且带动效应强
- **资金认可度**：成交放量、板块内多只个股跟涨
- **催化密度**：政策/事件催化持续释放

主线分层：
- **核心主线**：持续性强、辨识度高、资金集中
- **次线**：有主题催化但持续性待观察
- **短线支线**：事件驱动、一日游概率大

### 第三步：情绪周期定位

基于市场微观结构判断情绪所处阶段：

| 阶段 | 特征 | 操作含义 |
|------|------|---------|
| 冰点 | 跌停多、涨停少、成交缩量、连板高度压缩至2板 | 反弹临近，可试探性布局 |
| 修复 | 跌停减少、涨停回升、连板高度恢复 | 跟随龙头，控制仓位 |
| 主升 | 涨停爆发、赚钱效应扩散、成交放量 | 积极参与主线 |
| 退潮 | 高位分歧加大、补涨股开始活跃、龙头见顶 | 降低仓位，回避追高 |

判断依据：
- 涨停家数及连板高度
- 跌停家数
- 昨日涨停今日表现（溢价率）
- 成交额变化趋势

### 第四步：风格偏好

判断当前市场的风格倾向：

**数据获取**：
```bash
# 大盘vs小盘（沪深300 vs 中证1000/2000）
westock-data kline sh000300 day 10
westock-data kline sh000852 day 10

# 价值vs成长（中证红利 vs 科创50/创业板）
westock-data kline sh000922 day 10
westock-data kline sz399006 day 10
```

关注切换信号：
- 大小盘相对强弱连续 3 日反转
- 成交额结构变化（权重股放量 vs 小票放量）
- 北向资金风格偏好变化

### 第五步：板块强度排名

**数据获取**：
```bash
westock-data hot board 20
westock-data board
```

排名维度：涨幅、资金流入、连续强势天数、龙头辨识度。

## 注意事项

- 数据来源需标注（如 westock-data market、westock-data changedist 等）
- 主线判断需区分"客观数据"和"主观推断"
- 情绪周期是模糊判断，用"更接近X阶段"而非"确定在X"
- 盘中分析需注明时间节点，盘后复盘需注明是收盘数据
- 如数据接口异常，降级使用 WebSearch 搜索"A股行情 今日复盘"
